# @synth-interview/types

Shared TypeScript definitions and interfaces for the SynthInterview project.

## Usage

This package is used to maintain type safety across the monorepo, particularly between the frontend and common library utilities.

## Main Types

- **SessionData:** Represents the state and configuration of an interview session.
- **QuestionData:** Structure for coding problems, including descriptions and metadata.
- **ScorecardData:** The schema for generated evaluation reports.
- **RunResult:** Results from the code execution engine.
